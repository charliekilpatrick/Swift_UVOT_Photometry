SwiftPhotom.commands.uvotimsum(_in, _out, _exclude='none', ignoreframetime=False)[source]

Sum UVOT sky images into a single output file.

Parameters:
_instr

Input FITS file (optionally with extension specifier).

_outstr

Output FITS file path.

_excludestr, optional

Comma-separated extension indices to exclude, or 'none'.

ignoreframetimebool, optional

If True, combine extensions with different FRAMTIME (uvotimsum ignoreframetime=yes). Photometry on the sum may be less accurate.
